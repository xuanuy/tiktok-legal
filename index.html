Donghua Reels Tool

A personal content automation tool

This is a personal tool that prepares short vertical anime/donghua videos and uploads them to the owner's own TikTok account inbox via the TikTok Content Posting API. The owner reviews each video in the TikTok app, adds a sound, and publishes it manually.

What it uses

  • TikTok Content Posting API — scope video.upload, to upload the owner's own videos to the owner's own account inbox only.

It does not access, read, or post on behalf of other users, and uses no tracking or advertising.
